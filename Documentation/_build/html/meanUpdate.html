meanUpdate ¶

meanUpdate.meanUpdateM(model, M, P)[source]¶

Conserved state update for mean intensities

Parameters:
  • model (struct) – Pre-initialized model struct. See initializeModel.
  • M (vector) – Means for intensities of states. The first states must represent Quiescent (Q) neurons. The second states must represent actively firing (A) neurons. The remaining states can implement arbitrary linear dynamics (with Poisson noise).
  • P (matrix) – Precision matrix; Optional if using LNA
Returns:

M – Updated state estiamte for mean intensities
Return type:

vector
