util.nearestPositivePoint ¶

util.nearestPositivePoint.nearestPositivePoint(x0, P)[source]¶

Function nearestPositivePoint finds a point x based on the multivariate Gaussian with mean x0 and precision P such that x is the most probably point such that all entries in x are positive.

This is done using quadratic programming.

First, we transform into a coordinate system where Euclidian distance reflects distance in probability.

Then, we re-code the positivity constraint as a collection of constraints in this tranformed space, building an open simplex.
